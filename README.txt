== Installation ==

Now We want to describe how to install Ele Pack

* Go to Plugins
* Upload `ele-pack` folder to the `\wp-content\plugins` directory
* Install the plugin
* Activate the plugin
* Then go ELE PACK category you get addons to use
* Drag and drop and easy to use and customize
* Then let's enjoy it!
